[微信小程序开发]如何去掉或隐藏小程序顶部栏navigationBar

Rolan 2018-7-30 00:12

1033

小程序全局配置app.josn增加选项："navigationStyle": "custom",具体如下：原生小程序开发打开 app.json：{ "pages": , "window": { "backgroundTextStyle": "light", "navigationBarBackgroundColor": "#000", "

小程序全局配置app.josn增加选项："navigationStyle": "custom",

具体如下：

原生小程序开发

打开 app.json：

  1. {
  2.     "pages": [
  3.         "pages/index/index",
  4.         "pages/logs/index"
  5.     ],
  6.     "window": {
  7.         "backgroundTextStyle": "light",
  8.         "navigationBarBackgroundColor": "#000",
  9.         "navigationStyle": "custom",
  10.         "navigationBarTitleText": "WeChat"
  11.     }
  12. }

mpvue方式开发小程序>

打开 /src/main.js：

  1. export default {
  2.   config: {
  3.     pages: [
  4.       'pages/logs/main',
  5.       '^pages/index/main'
  6. 　　],
  7.     window: {
  8.       backgroundTextStyle: 'light',
  9.       navigationBarBackgroundColor: '#666',
  10.       navigationStyle: 'custom',
  11.       navigationBarTitleText: 'WeChat',
  12.       navigationBarTextStyle: '#fff',
  13.       enablePullDownRefresh: false,
  14.       disableScroll: true
  15.     }
  16.   }
  17. }
